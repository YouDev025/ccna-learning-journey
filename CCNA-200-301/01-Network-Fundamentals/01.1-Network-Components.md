# Network Components

## Definition
Network components are the devices, media, and services that make network communication possible. In CCNA, you should be able to identify what each component does and where it fits in a network design.

## Simple explanation
Think of a network as a delivery system. End devices create or consume the package, switches move it inside the local building, routers move it between buildings, and security devices inspect or control what is allowed through.

## Main categories
| Category | Examples | Main job |
|---|---|---|
| End devices | PCs, phones, printers, cameras | Create, send, receive, or use data |
| Intermediary devices | Switches, routers, APs, firewalls | Forward, connect, or protect traffic |
| Network media | Copper, fiber, wireless radio | Carry bits between devices |
| Network services | DHCP, DNS, NTP, AAA | Support addressing, names, time, and access |

## Important components
- Router: connects different IP networks and chooses paths.
- Layer 2 switch: connects devices inside the same LAN using MAC addresses.
- Layer 3 switch: switches like a Layer 2 switch and can route between VLANs.
- Firewall: filters traffic based on security policy.
- Wireless access point: connects wireless clients to the wired LAN.
- Controller: centrally manages devices such as APs or software-defined networks.
- Server: provides shared services or applications.
- Endpoint: user or IoT device where traffic starts or ends.
- PoE device: receives electrical power over an Ethernet cable.

## How traffic moves
1. A laptop creates traffic for a website.
2. The switch forwards the frame toward the default gateway.
3. The router forwards the packet toward another network.
4. Firewalls may inspect and permit or deny the traffic.
5. DNS, DHCP, and other services may support the connection.

## Exam focus
- Know the difference between end devices, intermediary devices, and media.
- Know that switches usually forward frames by MAC address.
- Know that routers forward packets by IP address.
- Know that firewalls enforce security policy, not just connectivity.

## Common mistake
Do not describe every network device as a router. A switch connects devices in a LAN; a router connects different networks.

## Quick review
Network components work together: endpoints create traffic, media carries it, switches and routers move it, and security devices protect it.
