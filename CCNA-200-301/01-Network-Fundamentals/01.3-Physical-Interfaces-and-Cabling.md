# Physical Interfaces and Cabling

## Definition
Physical interfaces and cabling provide the Layer 1 connection that carries bits between network devices.

## Common media types
| Media | Common use | Notes |
|---|---|---|
| Copper UTP | Ethernet to PCs, phones, APs | Uses RJ-45 connectors |
| Fiber optic | Uplinks, long distance, high speed | Uses light instead of electrical signals |
| Wireless | Wi-Fi clients | Uses radio frequencies |
| Console cable | Device management | Not used for normal data forwarding |

## Copper basics
- Common connector: RJ-45.
- Common cable types: Cat5e, Cat6, Cat6A.
- Typical Ethernet copper distance limit: 100 meters.
- Auto-MDIX handles straight-through/crossover needs on modern devices.

## Fiber basics
- Single-mode: longer distance, often campus or WAN links.
- Multimode: shorter distance, common in buildings and data centers.
- SFP/SFP+ transceivers connect fiber to network devices.

## Useful IOS commands
```text
show interfaces status
show interfaces
show ip interface brief
```

## Exam focus
Layer 1 problems often appear as down interfaces, errors, low speed, or duplex mismatches.

## Quick review
Physical connectivity must be healthy before higher-layer troubleshooting can succeed.
