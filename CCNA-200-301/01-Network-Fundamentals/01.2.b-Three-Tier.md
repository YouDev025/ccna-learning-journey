# Three-Tier Network Design

## Definition
A three-tier campus design separates the network into access, distribution, and core layers.

## Layers
| Layer | Main role |
|---|---|
| Access | Connects endpoints and provides edge features |
| Distribution | Aggregates access switches, applies policy, performs routing |
| Core | Provides fast, resilient transport between distribution blocks |

## Access layer features
- VLAN assignment.
- Port security and 802.1X.
- PoE for phones and APs.

## Distribution layer features
- Inter-VLAN routing.
- Route summarization.
- ACLs and policy boundaries.

## Core layer features
- High-speed forwarding.
- Redundant paths.
- Minimal policy filtering.

## Exam focus
The core should be fast and reliable. Heavy policy work usually belongs at the distribution layer.

## Quick review
Three-tier design separates access, distribution, and core functions for scalability and resilience.
