# Interface and Cable Issues

## Definition
Interface and cable issues are Layer 1 or Layer 2 problems that prevent a link from working correctly or cause poor performance.

## Common symptoms
- Interface is down/down.
- Interface is up/down.
- High CRC errors.
- High input errors.
- Late collisions or collisions.
- Slow performance.
- Flapping link.
- Wrong speed or duplex.

## Common causes
- Bad cable or connector.
- Cable too long.
- Disabled switch port.
- Wrong transceiver type.
- Speed or duplex mismatch.
- Damaged interface.
- Incorrect VLAN or trunk configuration.
- PoE power problem.

## Status examples
| Status | Likely meaning |
|---|---|
| administratively down | Interface is shut down by configuration |
| down/down | No physical link detected |
| up/down | Physical link exists but Layer 2 problem may exist |
| up/up | Interface is physically and logically active |

## Useful IOS commands
```text
show interfaces status
show interfaces
show ip interface brief
show running-config interface <interface>
show power inline
```

## Exam focus
CRC errors often point to physical media problems. Duplex mismatches can cause collisions and poor performance.

## Quick review
Start troubleshooting from Layer 1: cable, port, speed, duplex, and power.
