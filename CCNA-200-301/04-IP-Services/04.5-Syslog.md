# Syslog

## Definition
Syslog is a standard way for devices to send log messages to a centralized logging server.

## Simple explanation
It is like a notebook that devices use to record what happened.

## Why syslog is useful
- Centralizes logs
- Helps with troubleshooting
- Supports security monitoring

## Example
A firewall sends a syslog message when it blocks suspicious traffic.

## Quick review
Syslog stores and collects event messages from network devices.
