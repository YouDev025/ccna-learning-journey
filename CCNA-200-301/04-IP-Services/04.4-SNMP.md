# SNMP (Simple Network Management Protocol)

## Definition
SNMP is used to monitor and manage network devices.

## Simple explanation
It allows administrators to check device health and collect information from routers, switches, and servers.

## Main uses
- Monitor bandwidth and CPU usage
- Check device status
- Receive alerts when something goes wrong

## Example
An admin uses SNMP to see that a switch has high memory usage.

## Quick review
SNMP helps administrators manage devices remotely.
