# DHCP Relay

## Definition
A DHCP relay forwards DHCP requests from clients to a DHCP server on another subnet.

## Simple explanation
If a client and DHCP server are on different networks, the relay helps the request reach the server.

## Why it is used
- Helps DHCP work across subnets
- Reduces the need for a DHCP server in every subnet
- Makes centralized IP management easier

## Example
A client in VLAN 20 sends a DHCP request, and a router or relay agent forwards it to a DHCP server in VLAN 10.

## Quick review
DHCP relay bridges DHCP traffic between subnets.
