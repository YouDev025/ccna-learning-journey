# NAT (Network Address Translation)

## Definition
NAT is used to translate private IP addresses to a public IP address so devices can access the internet.

## Simple explanation
It is like giving your home devices a public address only when they need to go outside.

## Why NAT is used
- Saves public IPv4 addresses
- Hides internal private addresses
- Improves security by keeping internal addresses private

## Example
A laptop with IP address 192.168.1.10 connects to the internet through a router that translates it to a public IP.

## Quick review
NAT helps private devices communicate with public networks.
