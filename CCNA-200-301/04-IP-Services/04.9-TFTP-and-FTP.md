# TFTP and FTP

## Definition
FTP and TFTP are protocols used to transfer files between devices.

## Simple explanation
They are used when one device needs to upload or download files, such as configuration files or software images.

## FTP
- More secure than TFTP when used with authentication
- Supports directory listing and file transfer features

## TFTP
- Simpler and lighter
- Usually used for basic file transfers
- Less secure and less feature-rich

## Example
A network engineer uses TFTP to upload a new IOS image to a router.

## Quick review
FTP is more feature-rich, while TFTP is simpler and lighter.
