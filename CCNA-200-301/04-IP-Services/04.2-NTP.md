# NTP (Network Time Protocol)

## Definition
NTP is used to synchronize the clocks of devices on a network.

## Simple explanation
It makes sure all devices show the same time, which is important for logs and troubleshooting.

## Why NTP matters
- Keeps system logs accurate
- Helps with event correlation
- Supports secure authentication and certificates

## Example
A router, switch, and server all use NTP so their logs show the same timestamp.

## Quick review
NTP keeps network devices synchronized in time.
