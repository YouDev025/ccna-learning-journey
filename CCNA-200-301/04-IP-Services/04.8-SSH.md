# SSH (Secure Shell)

## Definition
SSH is a secure way to remotely access a device over a network.

## Simple explanation
It is like a protected remote-control connection for routers, switches, and servers.

## Why SSH is important
- Encrypts the session
- Provides secure remote administration
- Replaces insecure Telnet

## Example
An administrator connects to a switch using SSH to configure it from another location.

## Quick review
SSH is the secure method for remote device management.
