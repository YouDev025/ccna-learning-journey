# DHCP and DNS

## Definition
DHCP automatically assigns IP addresses to devices. DNS translates names like www.example.com into IP addresses.

## Simple explanation
DHCP gives a device an address, and DNS helps it find the right destination by name.

## DHCP
- Assigns IP address
- Gives subnet mask, gateway, and DNS server information
- Makes network setup easier

## DNS
- Converts names to IP addresses
- Used by browsers and applications
- Helps users access services without memorizing IPs

## Example
When a laptop joins a network, DHCP gives it 192.168.1.20. Then DNS helps it find the web server by name.

## Quick review
DHCP provides addressing; DNS provides name resolution.
