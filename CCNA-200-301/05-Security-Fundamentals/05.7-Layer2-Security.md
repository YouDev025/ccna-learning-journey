# Layer 2 Security

## Definition
Layer 2 security protects switching infrastructure from attacks or mistakes that target Ethernet, VLANs, trunks, MAC learning, and local network behavior.

## Simple explanation
Many attacks happen inside the LAN. Layer 2 security controls help stop rogue devices, spoofing, VLAN hopping, and switching loops.

## Common Layer 2 threats
- MAC address flooding
- VLAN hopping
- DHCP spoofing
- ARP spoofing
- Rogue switches
- STP manipulation
- Unauthorized devices on access ports

## Common protections
| Protection | Purpose |
|---|---|
| Port security | Limits allowed MAC addresses on a port |
| DHCP snooping | Blocks rogue DHCP servers |
| Dynamic ARP Inspection | Validates ARP messages using DHCP snooping data |
| BPDU Guard | Shuts down PortFast ports that receive BPDUs |
| Root Guard | Prevents unexpected switches from becoming STP root |
| Disable unused ports | Reduces attack surface |

## Port security example
```text
interface fastEthernet0/10
 switchport mode access
 switchport port-security
 switchport port-security maximum 2
 switchport port-security mac-address sticky
 switchport port-security violation shutdown
```

## DHCP snooping example
```text
ip dhcp snooping
ip dhcp snooping vlan 10,20
interface gigabitEthernet0/1
 ip dhcp snooping trust
```

## BPDU Guard example
```text
interface range fastEthernet0/1 - 24
 spanning-tree portfast
 spanning-tree bpduguard enable
```

## Good practices
- Set user ports to access mode.
- Disable DTP on access ports where supported.
- Use an unused native VLAN.
- Shut down unused ports and place them in an unused VLAN.
- Trust only uplink ports for DHCP snooping.

## Verification commands
```text
show port-security
show port-security interface <interface>
show ip dhcp snooping
show spanning-tree summary
```

## Exam focus
Know port security, DHCP snooping, Dynamic ARP Inspection, BPDU Guard, and why unused ports should be shut down.

## Quick review
Layer 2 security protects the local switching environment from spoofing, rogue devices, loops, and VLAN attacks.
