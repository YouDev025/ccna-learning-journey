# Wireless Security

## Definition
Wireless security protects Wi-Fi networks from unauthorized access, eavesdropping, and wireless-based attacks.

## Simple explanation
Because Wi-Fi uses radio waves, extra security controls are needed to prevent attackers from accessing the network from nearby locations.

## Common wireless security practices
- Use WPA2 or WPA3 instead of older insecure protocols
- Use strong and unique passwords for Wi-Fi access
- Separate guest users from internal devices with a guest SSID or VLAN
- Disable WPS if not needed
- Monitor for rogue access points

## Security considerations
- WPA3 offers stronger protection than WPA2 in many scenarios
- Enterprise authentication is more secure than shared PSK for business environments
- Guest networks reduce the risk of exposing internal resources

## Example
A company sets up WPA3 for employees and a separate guest network for visitors so guests cannot reach internal servers.

## Exam-style note
Wireless security should protect both the access point and the user traffic. A weak password or open guest network can create serious risk.

## Quick review
Wireless security protects the airwaves around the network by combining strong encryption, access control, and monitoring.
