# Security Concepts

## Definition
Security concepts are the principles and practices used to protect networks, systems, and information from unauthorized access, damage, or disruption.

## Simple explanation
Security is about making sure the right people can access the right resources while keeping attackers and accidental misuse away.

## Core security goals
- Confidentiality: protect data from being read by unauthorized users
- Integrity: ensure data is not altered without permission
- Availability: ensure systems and services remain available when needed

## Common security principles
- Least privilege: users should only have the access they need
- Defense in depth: use multiple layers of security instead of relying on one control
- Segmentation: separate sensitive systems from less secure areas
- Strong authentication: require reliable proof of identity

## Common threats
- Malware
- Phishing
- Unauthorized access
- Denial of Service (DoS)
- Insider threats

## Example
A company stores employee payroll files in a protected system so only HR managers can view them.

## Exam-style note
The CIA triad is one of the most important security foundations in networking and cybersecurity.

## Quick review
Security protects data, devices, and services by combining policies, technology, and monitoring.
