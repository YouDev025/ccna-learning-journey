# Security Concepts

## Definition
Security concepts are the principles and practices used to protect networks, systems, and information from unauthorized access, damage, disruption, or misuse.

## Simple explanation
Security is about making sure the right people can access the right resources, while attackers, mistakes, and unsafe traffic are kept away.

## CIA triad
| Goal | Meaning | Example |
|---|---|---|
| Confidentiality | Prevent unauthorized reading of data | Encryption and access control |
| Integrity | Prevent unauthorized changes to data | Hashing, checksums, file permissions |
| Availability | Keep services usable when needed | Redundancy, backups, DoS protection |

## Common security principles
- Least privilege: users and devices should have only the access they need.
- Defense in depth: use multiple layers of protection.
- Segmentation: separate sensitive systems from general user areas.
- Zero trust mindset: verify access instead of assuming internal traffic is safe.
- Strong authentication: prove identity with reliable methods.
- Monitoring and logging: detect suspicious activity and support investigations.

## Common threats
- Malware
- Phishing
- Unauthorized access
- Denial of Service (DoS)
- Man-in-the-middle attacks
- Insider threats
- Credential theft
- Misconfiguration

## Common controls
| Control type | Examples |
|---|---|
| Administrative | Policies, training, procedures |
| Technical | ACLs, firewalls, encryption, AAA |
| Physical | Locks, badges, cameras, secure rooms |

## Network security examples
- ACLs restrict traffic between VLANs.
- Firewalls inspect traffic between zones.
- WPA2/WPA3 protects wireless access.
- SSH protects device management sessions.
- AAA centralizes login control and auditing.

## Exam focus
The CIA triad is foundational. Also know least privilege, defense in depth, segmentation, authentication, authorization, and accounting.

## Quick review
Security protects data, devices, users, and services by combining policy, technical controls, monitoring, and good design.
