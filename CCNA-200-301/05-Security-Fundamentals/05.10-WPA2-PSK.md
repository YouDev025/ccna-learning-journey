# WPA2-PSK

## Definition
WPA2-PSK is a Wi-Fi security method that uses a shared pre-shared key (PSK) to authenticate users and encrypt wireless traffic.

## Simple explanation
All users who know the password can join the wireless network. It is easy to deploy but depends heavily on a strong password.

## How it works
- The client and AP share the same password
- A four-way handshake proves the client knows the password
- Data traffic is then encrypted using AES

## Advantages
- Easy to configure
- Common in homes and small offices
- Provides strong encryption for wireless traffic

## Limitations
- Not ideal for large enterprise networks
- The shared password can be compromised if weak or exposed
- Every user shares the same credential

## Example
A small office uses WPA2-PSK with a long, complex password so employees can connect securely to the Wi-Fi.

## Exam-style note
WPA2-PSK is suitable for simple environments, but WPA2-Enterprise is more secure for larger organizations.

## Quick review
WPA2-PSK uses a shared key and encryption to secure wireless access.
