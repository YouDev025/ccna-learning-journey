# IPsec VPN

## Definition
IPsec is a set of security protocols used to protect data transmitted over a VPN between devices or networks.

## Simple explanation
IPsec creates a secure tunnel so information can travel safely across untrusted networks such as the internet.

## Key features
- Encryption: protects the contents of traffic
- Authentication: verifies the identity of endpoints
- Integrity: ensures traffic has not been altered

## Common IPsec components
- IKE: used to establish secure negotiation and keys
- ESP: provides encryption and integrity for IP packets
- AH: provides integrity and authentication, but not encryption

## Modes
- Transport mode: protects traffic between two hosts
- Tunnel mode: protects traffic between networks and is commonly used in VPNs

## Example
A remote employee connects to the corporate network through an IPsec VPN before accessing internal resources.

## Exam-style note
IPsec is widely used for site-to-site and remote-access VPNs because it provides strong confidentiality and authentication.

## Quick review
IPsec secures data by encrypting and authenticating traffic inside a VPN tunnel.
