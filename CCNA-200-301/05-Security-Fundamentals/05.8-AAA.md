# AAA (Authentication, Authorization, and Accounting)

## Definition
AAA is a security framework used to manage user access and record activity on network devices and systems.

## Simple explanation
AAA answers three important questions: who is the user, what are they allowed to do, and what actions did they perform?

## Components of AAA
- Authentication: verifies the identity of the user
- Authorization: determines what the user is permitted to access
- Accounting: logs actions for auditing and monitoring

## Common AAA protocols
- RADIUS: commonly used for network access and remote authentication
- TACACS+: often used for device administration because it separates authentication, authorization, and accounting more clearly

## Example
A network engineer logs into a router using AAA. The system checks their credentials, grants only administrator rights, and records the session in logs.

## Exam-style note
AAA is especially useful in centralized security environments because it allows consistent policy enforcement across devices.

## Quick review
AAA improves security by controlling access and maintaining accountability.
