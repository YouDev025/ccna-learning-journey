# Packet Tracer Labs

This folder contains practical Cisco Packet Tracer labs for CCNA 200-301 practice.

## Structure

- `Beginner/` - foundational topology, addressing, and device access labs.
- `Intermediate/` - VLANs, routing, DHCP, NAT, and common services.
- `Exam-Level/` - timed troubleshooting and mixed-topic challenge labs.

## Suggested Lab Workflow

1. Read the objective and topology.
2. Build the topology in Cisco Packet Tracer.
3. Configure devices from the requirements.
4. Verify connectivity with `ping`, `traceroute`, `show ip interface brief`, and related `show` commands.
5. Save the `.pkt` file beside the lab notes.
6. Record lessons learned and commands used.

## Naming Convention

Use this format for Packet Tracer files:

```text
NN-lab-topic-name.pkt
```

Example:

```text
01-basic-device-connectivity.pkt
```
