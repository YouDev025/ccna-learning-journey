# Lab 03: Static Routing

## Objective

Configure static routes between multiple routers and verify end-to-end connectivity.

## Topology

- 3 routers
- 3 LANs
- 3 PCs

## Tasks

1. Build a three-router topology.
2. Configure all router interfaces.
3. Configure static routes on each router.
4. Configure PC IP addresses and default gateways.
5. Verify connectivity from each LAN to every other LAN.

## Useful Commands

```text
ip route
show ip route
show ip interface brief
ping
traceroute
```
