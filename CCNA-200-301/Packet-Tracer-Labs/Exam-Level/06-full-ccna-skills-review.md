# Lab 06: Full CCNA Skills Review

## Objective

Complete a broad CCNA-style configuration challenge from requirements.

## Required Features

- VLANs
- Trunks
- Inter-VLAN routing
- DHCP
- Static routing or OSPF
- NAT overload
- SSH management
- Basic device hardening
- Connectivity verification

## Tasks

1. Design the topology from the requirements.
2. Create an addressing table before configuring devices.
3. Configure switching features.
4. Configure routing and services.
5. Secure management access with SSH.
6. Verify all required connectivity.
7. Save all device configurations.
8. Write a short summary of what you configured.

## Verification

```text
show running-config
show ip interface brief
show vlan brief
show interfaces trunk
show ip route
show ip dhcp binding
show ip nat translations
ping
traceroute
```
