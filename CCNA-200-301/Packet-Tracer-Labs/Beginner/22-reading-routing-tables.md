# Lab 22: Reading Routing Tables

## Objective

Learn how to read basic routing table entries.

## Topology

- 2 routers
- 2 switches
- 2 PCs

## Tasks

1. Build two LANs connected by two routers.
2. Configure IP addresses on all router interfaces.
3. Configure one static route on each router.
4. Display the routing table.
5. Identify connected, local, and static routes.
6. Test connectivity between the LANs.

## Useful Commands

```text
show ip route
show ip interface brief
ping
traceroute
```

## Questions

- What does `C` mean in the routing table?
- What does `L` mean in the routing table?
- What does `S` mean in the routing table?
