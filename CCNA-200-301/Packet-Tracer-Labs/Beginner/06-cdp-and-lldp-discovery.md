# Lab 06: CDP and LLDP Discovery

## Objective

Use neighbor discovery protocols to identify connected Cisco devices.

## Topology

- 2 routers
- 2 switches
- Ethernet links between devices

## Tasks

1. Build a small network with routers and switches.
2. Configure hostnames on all devices.
3. Verify CDP neighbor information.
4. Enable LLDP globally.
5. Verify LLDP neighbor information.
6. Compare CDP and LLDP output.

## Useful Commands

```text
show cdp neighbors
show cdp neighbors detail
lldp run
show lldp neighbors
show lldp neighbors detail
```
