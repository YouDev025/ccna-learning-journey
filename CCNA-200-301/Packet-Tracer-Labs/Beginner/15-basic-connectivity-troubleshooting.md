# Lab 15: Basic Connectivity Troubleshooting

## Objective

Troubleshoot common beginner-level connectivity problems.

## Topology

- 1 router
- 1 switch
- 2 PCs

## Possible Faults

- Wrong IP address
- Wrong subnet mask
- Missing default gateway
- Router interface shutdown
- Incorrect cable
- PC connected to the wrong switch port

## Tasks

1. Build the topology.
2. Configure the network with one or more intentional mistakes.
3. Use verification commands to find the issue.
4. Fix each issue.
5. Confirm successful connectivity.
6. Write down what caused the problem.

## Verification

```text
ipconfig
ping
traceroute
show ip interface brief
show running-config
```
