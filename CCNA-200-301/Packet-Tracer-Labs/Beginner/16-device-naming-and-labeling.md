# Lab 16: Device Naming and Labeling

## Objective

Practice naming devices clearly and labeling a small network topology.

## Topology

- 1 router
- 2 switches
- 4 PCs

## Tasks

1. Build a small two-switch LAN.
2. Rename devices in Packet Tracer using a clear naming scheme.
3. Configure hostnames on the router and switches.
4. Add topology labels for LAN names and IP networks.
5. Save the topology with all labels visible.

## Suggested Names

| Device | Hostname |
| --- | --- |
| Router | R1 |
| Switch 1 | SW1 |
| Switch 2 | SW2 |
| PCs | PC1, PC2, PC3, PC4 |

## Useful Commands

```text
hostname R1
hostname SW1
show running-config
```

## Notes

- Save the completed Packet Tracer file as `16-device-naming-and-labeling.pkt`.
