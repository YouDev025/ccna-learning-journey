# Lab 23: Basic Ping and Traceroute

## Objective

Use ping and traceroute to test network reachability and path selection.

## Topology

- 2 routers
- 2 switches
- 2 PCs

## Tasks

1. Build a two-router network.
2. Configure IP addresses and static routes.
3. Ping the default gateway from each PC.
4. Ping from one PC to the other PC.
5. Use traceroute to view the path between PCs.
6. Shut down one router interface and repeat the tests.

## Verification

```text
ping
traceroute
show ip interface brief
```

## Questions

- What does a successful ping prove?
- What does traceroute show that ping does not?
