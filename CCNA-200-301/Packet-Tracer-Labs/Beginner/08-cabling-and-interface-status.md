# Lab 08: Cabling and Interface Status

## Objective

Practice choosing correct cable types and checking interface status.

## Topology

- 1 router
- 1 switch
- 2 PCs

## Tasks

1. Connect PCs to the switch.
2. Connect the switch to the router.
3. Use the correct cable types.
4. Configure basic IP addresses.
5. Check interface status on the router and switch.
6. Fix any disconnected or down interfaces.

## Useful Commands

```text
show ip interface brief
show interfaces status
show interfaces
```

## Questions

- Which interfaces are up?
- Which interfaces are administratively down?
- What command enables a router interface?
